  2. <meter>

    Syntax : <meter> ....... </meter>

    It defines a scaler measurement within a known range. This is also known as gauge. It is a paired tag.

    Attributes:

    Attributes Value Description
    form form_id Specifies one or more forms.
    high number Specifies the range that is considered to be high.
    low number Specifies the range that is considered to be low
    max number Specifies the maximum value of the range.
    min number Specifies the minimum value of the range.
    optimum number Specifies what value is the optimum value for gauge.
    value number Specifies the current value for the gauge.

  3. <progress>

    Syntax : <progress> ....... </progress>

    It is used to represent progress of a task. It is a paired tag.

    Attributes :

    Attributes Value Description
    max number Specifies how much work the task requires in total.
    value number Specifies how much of the task has been completed.

    Example:

  4. <time>

    Syntax: <time> ....... </time>

    This element is used to represent date and time in machine readable format.It is a paired tag.

    Attributes:

    Attribute Value Description
    datetime datetime Gives the date/time being specified.

    Example:
